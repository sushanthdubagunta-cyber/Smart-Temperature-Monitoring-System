# Smart Temperature Monitoring System

This project is a Smart Temperature Monitoring System developed as part of an Embedded Systems & IoT internship. The system uses an Arduino UNO and TMP36 temperature sensor to continuously monitor ambient temperature and generate visual and audible alerts when the temperature exceeds a predefined threshold.

## Features
- Real-time temperature monitoring
- Temperature display through Serial Monitor
- LED alert indication
- Buzzer alarm for high temperature detection
- Easy to simulate using Tinkercad or Cirkit Designer

## Components Used
- Arduino UNO
- TMP36 Temperature Sensor
- LED
- 220Ω Resistor
- Piezo Buzzer
- Jumper Wires

## Technologies
- Arduino IDE
- Embedded C/C++
- Tinkercad Simulation
- Cirkit Designer

## Applications
- Smart Home Monitoring
- Industrial Safety Systems
- Server Room Monitoring
- Greenhouse Temperature Control
- Environmental Monitoring
